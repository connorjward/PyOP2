class pyop2.codegen.node.Node¶

Bases: object

Abstract node class.

Nodes are not meant to be modified.

A node can reference other nodes; they are called children. A node might contain data, or reference other objects which are not themselves nodes; they are not called children.

Both the children (if any) and non-child data (if any) are required to create a node, or determine the equality of two nodes. For reconstruction, however, only the new children are necessary.

reconstruct(*args)¶

Reconstructs the node with new children from ‘args’. Non-child data are copied from ‘self’.

Returns a new object.

Returns a pymbolic call for C-based targets, when the instructions involve multiple return values along with the required type casting. The first assignee is returned, but the rest of them are appended to the parameters and passed by reference.

Example: c, d = f(a, b) is returned as c = f(a, b, &d)

Parameters

  • insn – An instance of loopy.kernel.instructions.CallInstruction.

  • target – An instance of loopy.target.TargetBase.

  • expression_to_code_mapper – An instance of IdentityMapper responsible for code mapping from loopy syntax to the target syntax.
